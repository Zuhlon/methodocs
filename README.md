## Ключевые концепции

### LD2 (Loyalty-Driven Product Design)
Методология проектирования продукта через сознательное выращивание лояльного ядра аудитории (15-25%), вместо попыток удовлетворить всех пользователей.

### Динамические артефакты
Замена статических документов (персон, CJM, User Flow) на самообновляемые предиктивные модели:
- K-Means кластеризация по поведению
- PrefixSpan для извлечения правил
- Предиктивный Score лояльности
- AI-ассистенты ядра

### Сегментация LD2
- **Ядро** (≥0.70) — лояльные пользователи, приоритет
- **Симпатики** (0.50-0.69) — потенциал перехода в ядро
- **Нейтралы** (0.30-0.49) — минимум инвестиций
- **Чужые** (<0.30) — игнорировать
